# Опрос пользователей (Django REST Framework)

## Как запустить
### Первоначальная установка
```
pip install -r requirements.txt
cd poll-app
python manage.py migrate
python manage.py createsuperuser
```

### Запуск сервера
```
cd poll-app
python manage.py runserver
```
